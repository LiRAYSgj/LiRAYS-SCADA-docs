# Users Guide

This section is for operators and final users who need to deploy and run LiRAYS-SCADA in lab or production environments.

## What You Can Do Here

- Install LiRAYS-SCADA using Docker, Linux packages, or macOS packages.
- Configure runtime behavior through `settings.yaml` or environment variables.
- Follow practical tutorials to get from installation to first operational checks.
