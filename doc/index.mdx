This site is the official documentation for LiRAYS-SCADA. It is organized to help you install, operate, integrate, and extend the platform with a consistent structure.

LiRAYS-SCADA is currently in its initial release stage. We strongly recommend testing it in your environment and sharing feedback so we can improve stability, usability, and deployment guidance.

## Documentation Sections

- [Users](/users): installation, configuration, and operator tutorials.
- [Integrators](/integrators): SDK and protocol integration references.
- [Developers](/developers): architecture, environment setup, and contribution workflow.
- [Product Roadmap](/roadmap): current planning direction and delivery timeline.

## Recommended Reading Path

1. Start in [Users](/users) if you are deploying or operating LiRAYS-SCADA.
2. Continue with [Integrators](/integrators) if you are connecting external software.
3. Use [Developers](/developers) if you are contributing to the codebase.
