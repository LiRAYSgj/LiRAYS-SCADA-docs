# LiRAYS-SCADA Documentation

This site is the official documentation for LiRAYS-SCADA. It is organized to help you install, operate, integrate, and extend the platform with a consistent structure.

## Documentation Sections

- [Users](users/index.md): installation, configuration, and operator tutorials.
- [Integrators](integrators/index.md): SDK and protocol integration references.
- [Developers](developers/index.md): architecture, environment setup, and contribution workflow.
- [Product Roadmap](roadmap/index.md): current planning direction and delivery timeline.

## Recommended Reading Path

1. Start in [Users](users/index.md) if you are deploying or operating LiRAYS-SCADA.
2. Continue with [Integrators](integrators/index.md) if you are connecting external software.
3. Use [Developers](developers/index.md) if you are contributing to the codebase.
