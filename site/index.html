LiRAYS-SCADA Documentation

This site is the official documentation for LiRAYS-SCADA. It is organized to help you install, operate, integrate, and extend the platform with a consistent structure.

Documentation Sections

  • Users: installation, configuration, and operator tutorials.
  • Integrators: SDK and protocol integration references.
  • Developers: architecture, environment setup, and contribution workflow.
  • Product Roadmap: current planning direction and delivery timeline.

Recommended Reading Path

  1. Start in Users if you are deploying or operating LiRAYS-SCADA.
  2. Continue with Integrators if you are connecting external software.
  3. Use Developers if you are contributing to the codebase.
